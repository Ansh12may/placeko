# placeko
Placeko - AI-Powered Resume Screening and Job Matching

An end-to-end AI-driven recruitment platform that automates resume parsing, feature extraction, and job matching using state-of-the-art language models, Named Entity Recognition (NER), and machine learning. The system leverages Retrieval-Augmented Generation (RAG), OpenAI LLMs, and spaCy to deliver high-accuracy semantic matching and transparent recruitment insights.

Features
Real-time parsing of resumes in multiple formats (PDF, DOCX, TXT)

Semantic matching of candidate profiles with job descriptions using embedding vectors and FAISS

Actionable feedback for candidates on resume improvements and recruiter-friendly analytics

Bias-minimized, fair ranking for improved hiring outcomes

Job data scraping integrated with SerpApi for live listings

Interactive Streamlit UI for candidates and recruiters with analytics dashboards

Results
Achieves 93-95% matching accuracy with significant reductions in screening time compared to traditional methods.


Tech Stack
Python, OpenAI API, spaCy, FAISS, Streamlit, SerpApi
